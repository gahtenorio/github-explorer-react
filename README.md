## 🚀 Technologies

This project was developed with the following technologies:

- ✔️ ReactJS
- ✔️ TypeScript
- ✔️ Styled Components
- ✔️ Axios

<br>
<br>

## Installation


```bash
$ git clone https://github.com/gahtenorio/github-explorer-react.git
$ cd github-explorer-react
$ yarn
```

## Run

```bash
$ yarn start
```
